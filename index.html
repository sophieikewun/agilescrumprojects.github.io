Price Compare

This project focused on developing an application to compare prices across UK stores, offering users two search options and direct links to the cheapest stores' websites. A Gantt chart was used to plan the project timeline, allocate duties among team members, and track progress effectively. The design process began with low-fidelity wireframes, progressing to medium- and high-fidelity prototypes to refine the user experience. Stakeholder feedback was integrated at each stage to ensure the application met user needs. The result: a user-friendly tool that simplifies price comparison, delivering value through efficiency and ease of use.

Smart Road Control

This project focused on creating software to enhance road safety and efficiency by enabling vehicles to connect with mobile data antennae for real-time traffic management. The system transmitted data between vehicles, regional control computers, and system controllers, ensuring optimized driving instructions and reducing accidents. The software relied on self-driving capabilities of modern vehicles and included a web service for staff to monitor and manage traffic flow. Stakeholder collaboration and iterative development ensured the system met user needs while addressing risks effectively.
